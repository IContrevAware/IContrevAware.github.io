MC Java

This guide explains how to install MC Java.

Step 1

Go to [Prism Launcher](prismlauncher.org).

Step 2

Download setup (or portable).

Step 3

Run installer or extract portable zip.

Step 4

Sign in with MS account

Step 5

Create instance and select mod loader

Step 6

Add performance, QoL or any mods.

Step 7

Launch.
